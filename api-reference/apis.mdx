**Authentication**

All REST endpoints are authenticated using your
[Browserbase API Key](/quickstart#start-a-browser-session-from-the-code),
configured as follow:

<CodeGroup>

```js TypeScript
const response = await fetch(
  `https://www.browserbase.com/v1/sessions/${sessionId}`,
  {
    method: "GET",
    headers: {
      "x-bb-api-key": `${process.env.BROWSERBASE_API_KEY}`,
    },
  },
);
```

```python Python
import requests
import os

url = "https://www.browserbase.com/v1/sessions"

headers = {"x-bb-api-key": os.environ["BROWSERBASE_API_KEY"]}

response = requests.request("GET", url, headers=headers)
```

```bash cURL
curl -X GET "https://www.browserbase.com/v1/sessions/${SESSION_ID}" \
     -H "x-bb-api-key: ${BROWSERBASE_API_KEY}" \
```
